== Description ==
VT Blogging is a Minimal and Clean WordPress Theme. Mobile-friendly (responsive), optimized for speed, and implements SEO (search engine optimization) best practices. The theme options are powered by Customizer. You can check out the demo for a closer look at: http://volthemes.com/demo?theme=VT-Blogging

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select 'vt-blogging.zip'. Click Install Now.
3. Click Activate to use your new theme right away.

- via FTP: Upload the theme folder to your "/wp-content/themes/" directory.

== Translation ==
